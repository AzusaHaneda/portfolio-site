#　 ポートフォリオサイト

## 概要
Web制作の学習の一環として、ChatGPTを活用しながら制作しました

## 使用技術
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 制作のポイント
- 初めてのWebサイト制作のため、基本的な構造（ヘッダー・フッター・2カラムレイアウト）を意識しました
- AIを活用してエラー解決やコードの書き方を学び、自分なりに調査・修正を行いました
- レスポンシブ対応（スマホでも見やすいレイアウト）を目指しました

## 学んだこと
- ファイルの配置方法
- Githubへのアップロード

## 今後の改善点
- AIと伴走しながらデザイン性を高める
- javascriptを活用して動きのあるサイトにする
